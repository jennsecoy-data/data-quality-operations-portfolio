## Example Quality Review Process

A typical review process includes:

1. Check records or documents against the required format.
2. Confirm that required fields are complete.
3. Identify missing, inaccurate, or inconsistent information.
4. Document issues clearly.
5. Track follow-up actions.
6. Summarize findings in a report or update.
7. Recommend process improvements when repeated issues appear.
